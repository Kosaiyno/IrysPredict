Welcome to IrysPredict

This is a simple, fast prediction game built on Irys datachain. Pick whether an asset will go up or down and earn points if you’re right. Rounds last 5 minutes. To keep it fair, betting is only open during the first 1 minute of each round. After that, the round plays out and you’ll see how your call did.

How points work

  • +10 for every correct prediction (win)
  • −6 for a wrong prediction (loss)
  • Streak boost: each win in a row adds +2 (capped at +10)
  • Daily activity: first 10 rounds you play today give +1 each (max +10)
  • Anti-spam tax: after 20 rounds today, you only earn 50% until tomorrow
  • One global streak: any loss resets it

Example: 3 wins in a row = 10 + (10+2) + (10+4) = 36 points.

A loss after that: −6 and streak resets.
